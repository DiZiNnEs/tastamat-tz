# Тестовое задание

Тестовое задание для Tastamat

## Установка
### Запуск web-application
**Список зависимостей для веб-приложения**:
```
Node.js
NPM / Yarn / Pnpm
```

**Запуск**:
```
npm i
ionic serve
```


### Запуск Android:
**Список зависимостей для веб-приложения**:
```
Node.js
NPM / Yarn / Pnpm
Java 17
Gradle v8
Android Studio 2023
```

**Запуск**:
```
npm i @capacitor/android
npx cap add android
npx cap open android
```
Более подробней - [ТЫК](https://capacitorjs.com/docs/android)

### Запуск iOS:
#### Сорри, MacOS нету : )

**Список зависимостей**:
